# SpaceX_React_App
### This is a React based application to show the details of all the launch done by spaceX

To run this project
cd into project
yarn install
yarn start

## Tech stack
create-react-app
axios
jest
enzyme

## Responsiveness
This app is responsive to mobile, tablet and desktop view
